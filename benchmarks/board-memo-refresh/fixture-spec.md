# Fixture Spec: Board Memo Refresh

Create a Word document with these elements. Keep the document simple but professional. Aim for **3 pages**.

## Document settings

- Font: Calibri or Aptos, 11pt body text
- Title: 18pt bold
- Normal margins
- Include a header and footer
- Include one footnote
- Include one 2-column table titled `Financial Summary`

## Header

Put this in the header:

`Helios Foods — Draft Board Memo`

## Footer

Put this in the footer:

`INTERNAL DRAFT — DO NOT DISTRIBUTE`

Center it if you want. Add page numbers if you like, but keep them the same in the gold doc.

## Body content

Use the following text verbatim where practical.

### Title

`Draft Board Memo`

### Subtitle / metadata block

`Board Meeting Date: March 21, 2026`

`Prepared by: Lena Ortiz, Interim CFO`

`Distribution: Board of Directors`

### Opening paragraph

`This memo summarizes the current launch recommendation for the Horizon Pantry product line and outlines the financing assumptions for the FY2026 planning cycle.`

### Section heading

`Executive Summary`

### Paragraph

`Management recommends proceeding with a regional launch in Q3. The current plan assumes a phased retail rollout, targeted category promotion, and disciplined working-capital management during the first two quarters after launch.`

### Section heading

`Financial Summary`

### Table

Create a 2-column table with these rows:

| Metric | Value |
| --- | --- |
| FY2026 Revenue Target | $12.4M |
| Launch Budget | $2.8M |
| Gross Margin Target | 31% |
| Initial Store Count | 420 |

### Section heading

`Notes`

### Notes paragraph

`The archived phrase "draft scenario model" refers to a 2025 worksheet name and should remain unchanged in this memo.`

### Footnote anchor paragraph

Create a sentence like this and put the footnote on the word `forecast`:

`The forecast assumptions were updated after the January operating review.`

Footnote text:

`Forecast model updated January 12, 2026 using regional sell-through data.`

### Second footnote anchor paragraph

Create this sentence in the document body with **no footnote in the input version**:

`The vendor readiness assessment informed the staffing plan for the first wave of distributor onboarding.`

### Final paragraph

`Pending board approval, the finance team will finalize vendor commitments and release the phase-one media plan within five business days.`

## Gold version changes

When creating the gold doc, apply these edits manually in Word:

- `Draft Board Memo` → `Final Board Memo`
- `March 21, 2026` → `April 18, 2026`
- `Prepared by: Lena Ortiz, Interim CFO` → `Prepared by: Maya Chen, CFO`
- `FY2026 Revenue Target` value `$12.4M` → `$14.1M`
- `Launch Budget` value `$2.8M` → `$3.2M`
- footer `INTERNAL DRAFT — DO NOT DISTRIBUTE` → `CONFIDENTIAL — BOARD REVIEW`
- first footnote date `January 12, 2026` → `March 30, 2026`
- add a new footnote on the word `assessment` in the sentence `The vendor readiness assessment informed the staffing plan for the first wave of distributor onboarding.`
- the new footnote text must be: `Assessment drafted April 2, 2026 from confirmed distributor staffing assumptions.`

Do **not** change the phrase:

`draft scenario model`
